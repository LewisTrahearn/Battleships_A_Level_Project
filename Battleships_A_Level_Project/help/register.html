Battleships Game Register Help

How to Register in the system.

Read the instructions below or watch the video on how to register in the battleships game.

Please note that if you are already registered then you only need to login to the battleships game login

in order to Register follow the instructions below or watch the video at the end of this page.

Instructions:

1 Splashscreen

from the splashscreen select the Register option

2 Register Screen

You should now see the following screen. The messages on the right are only there because I clicked the submit button without entering correct details

3 Data Entry

you should now enter the information according to what each field requires.

  1. forename - a text input of 21 characters
  2. surname - a text input of 21 characters
  3. Username -an alphanumeric input of 21 characters, this is the name that will be displayed whilst playing the game
  4. email - this must be a valid email address. The format will be validated, an invalid format or address will stop you registering
  5. Password - an alphanumeric input of 21 characters
  6. confirm password - to ensure you have inputted the password as you would like.

4 Submit

Once both these fields have been entered then click on the Submit button

If all the fields are correct then you will be taken to the Options Screen, otherwise you will need to enter these again. Use the [return] key to cycle between the fields and the [DEL] and [BACKSPACE] keys to alter the fields.
