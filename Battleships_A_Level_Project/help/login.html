Battleships Game Login Help

How to login to the system.

Read the instructions below or watch the video on how to log into the battleships game.

Please note that in order to login, you must first have registered yourself within the system. If you have not done this then please select the following link and start by registering Register

in order to login follow the instructions below or watch the video at the end of this page.

Instructions:

1 Splashscreen

from the splashscreen select the login option

2 Login Screen

You should now see the following screen

3 Username

The cursor should now be positioned and flashing within the user name field. If not click on the screen with you mouse to ensure the application has focus set.

With the cursor flashing start typing the user name. You will notice that in this case I have entered the name my_username. The username must be a single word without whitespace.

4 Password

Once you have entered the username, press the [return] key and the cursor will be positioned in the password field.

Enter the password in the same way as the username.

5 Submit

Once both these fields have been entered then click on the Submit button

If the username and password were correct then you will be taken to the Options Screen, otherwise you will need to enter these again. Use the [return] key to cycle between the username and password and the [DEL] and [BACKSPACE] keys to alter the fields.
