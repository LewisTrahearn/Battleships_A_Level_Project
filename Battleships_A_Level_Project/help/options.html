Battleships Game Options Screen

How to Register in the system.

Read the instructions below or watch video on how to navigate the options screen.

Please note that if you are already registered then you only need to login to the battleships game login

in order to Register follow the instructions below or watch the video at the end of this page.

Instructions:

1 Options Navigation

From this screen you will be able to complete the following:

  1. Play against the computer
  2. Play against an opponent (a second player)
  3. Edit your profile or registration details
  4. View the ships log (i.e. see player and game statisics)
  5. Show HTML help for the game.

2 simulation

This screen lets you practice and play against the computer. Once you have selected this option on the leftbar, press the play button and it will take you to the game screen

3 Opponent

This screen lets you practice and play against A human opponent. Once you have selected this option on the leftbar, you need to select the login option to log the second person in then press the play button and it will take you to the game screen

4 Profile

This screen lets you edit your registered profile however you must be the main user i.e. player 1. If player 2 wishes to update their profile then player 1 must log out of the game and player 2 must log.

5 Ships Log

This screen lets you track the stats and allows you to see your personal best, where you lie in comparison to other players and the top 10 of the best times and lowest number of hits.

6 Help

Selecting this option will launch the HTML help screen in your browser.

7 Quit

This button will return you to the splashcreen and log out both players
